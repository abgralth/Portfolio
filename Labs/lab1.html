Assignement 1

a) HTML: HTML is a web language used to write some part of a page.

b) CSS: CSS is a language used in web design to modify easily the look of a page and make it more appealing.

c)JavaScript: JavaScript is a language used to add more complexity to a page in order to improve the design even more than CSS.

d)HTTP: HTTP is a protocol which allow a communication between the computer and the server.

e) HTML5: HTML5 is simply the newest version of HTML which we are most likely going to use.

f) Dynamic HTML: A HTML page is called dynamic when it can be modified on the run while browsing the page.
